# Calculadora basica em C

Calculadora simples desenvolvida em C com menu interativo para as quatro operações básicas: adição, subtração, multiplicação e divisão. O programa valida a divisão por zero e permite repetir as operações.

## Funcionalidades

- Menu com as opções:
  - 1 - Adição
  - 2 - Subtração
  - 3 - Multiplicação
  - 4 - Divisão
- Leitura de dois números reais (`float`).
- Exibição do resultado com duas casas decimais.
- Tratamento de erro para divisão por zero.
- Opção de realizar outra operação ou encerrar.
